Step4 - Polya定理-例题&&模板（POJ 1286）

题目大意：

给你一个长度为 n 的项链（环），着 3 种颜色，问有多少种

解题思路：

  1. 确定置换个数：旋转 (k * 360 / n)° n 种， 对称反转 n 种， 共 2 * n 种
  2. 确定循环节个数，旋转角度 (k * 360 / n)° 循环节为 gcd(n, k)，对称反转分奇数和偶数。
  3. 套用 Polya 公式求解
