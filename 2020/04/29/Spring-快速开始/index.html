xml文件中头部的一些元素的解释

  • xmlns：默认命名空间
  • xmlns:xsi：业界默认的用于 XSD(（XML Schema Definition) 文件的命名空间
  • xsi:schemaLocation：XSD文件的位置
  • XSD文件：XSD 文件（也常常称为 Schema 文件）是用来定义 xml 文档结构的
